# coreos-kvm-automation  

Automate the creation of CoreOs vms on KVM

## Prerequisites  

- ruby  
- git  
- qemu-img  

## Usage  

```bash
git clone --depth 1 https://github.com/camilin87/coreos-kvm-automation.git
cd coreos-kvm-automation
ruby create_coreos_vm.rb
```
